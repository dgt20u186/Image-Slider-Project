# Слайдер изображений

Адаптивный слайдер с автопрокруткой, управлением с клавиатуры, сенсорными свайпами и индикаторами (точками).

## Возможности
- Плавная анимация fade
- Автоматическое переключение каждые 5 секунд (останавливается при наведении)
- Кнопки "вперёд/назад"
- Индикаторы-точки с возможностью перехода по клику
- Управление стрелками клавиатуры
- Поддержка свайпов на мобильных устройствах
- Адаптивный дизайн
